# To Do List (Lista de tarefas)



# Descrição 
Meu primeiro projeto de estudo sobre JavaScript.
<br/>
Criação de uma lista de tarefas simples.

# Como funciona

* Pode escrever algo na lista 
* Pode adicionar algo na lista
* Pode excluir algo da lista
